## **Добавить тип устройства**

### Запрос
:::danger Пользователь должен быть разработчиком или администратором
:::

```
POST api/e/devices/
*bearer token*
{
    "device":{
        "name":"someType"
    }
}
```
### Ответ
Id нового типа устройства

## **Получить список всех типов устройств**
### Запрос
:::danger Пользователь должен быть разработчиком или администратором
:::

```
GET api/e/devices/
*bearer token*
```
### Ответ
Список всех типов устройств и их Id
